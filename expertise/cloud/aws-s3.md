Bucket policies, presigned URLs, and lifecycle rules. Bucket policy vs IAM policy for access control, presigned URL generation for secure upload/download with TTL, multipart upload for large files, lifecycle rules for transitioning to Glacier or expiring objects, versioning for data protection, server-side encryption (SSE-S3, SSE-KMS), and CORS configuration for browser uploads.
