Sanitization, schema validation, and injection prevention. Schema-based validation with Joi or Zod at API boundaries, SQL injection prevention via parameterized queries (never string concatenation), XSS prevention through output encoding, request body size limits, file type validation by magic bytes (not extension), and whitelist over blacklist approach.
