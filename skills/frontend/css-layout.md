Flexbox for one-dimensional layouts (nav bars, card rows), CSS Grid for two-dimensional layouts (page structure, dashboards), mobile-first responsive design with min-width breakpoints, fluid typography with clamp(), container queries for component-level responsiveness, and logical properties (inline/block) for internationalization support.
